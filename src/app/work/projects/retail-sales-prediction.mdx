## Project Overview

A machine learning project that forecasts daily sales for Rossmann drugstore branches. Using historical sales data, the model predicts future demand to support inventory planning and data-driven decisions. It was built end to end — from raw data to an interactive dashboard.

## What I Built

### Data preparation & feature engineering
- Cleaned the Rossmann dataset and handled missing values and outliers
- Engineered time-based features (day, month, promotions, holidays, competition distance) to capture seasonal and promotional patterns

### Modeling
- Trained an XGBoost regression model to predict store-level daily sales
- Tuned hyperparameters with grid search and validated with cross-validation
- Ran feature-importance analysis to understand the main sales drivers

### Evaluation
- Evaluated the model with regression metrics suited to forecasting: RMSE, MAE and R²

### Interactive dashboard
- Deployed a Streamlit app so anyone can explore the predictions interactively

## Technical Stack

- **Language**: Python
- **ML & data**: XGBoost, Pandas, NumPy, Scikit-learn
- **Visualization**: Matplotlib, Seaborn
- **App**: Streamlit
- **Environment**: Jupyter Notebook

## 🇹🇷 Proje Detayları

Rossmann ilaç mağazası şubelerinin günlük satışlarını tahmin eden bir makine öğrenimi projesi. Model, geçmiş satış verilerini kullanarak gelecekteki talebi tahmin eder ve envanter planlaması ile veri odaklı kararları destekler. Ham veriden interaktif panoya kadar uçtan uca geliştirildi.

### 📊 Veri Hazırlığı ve Özellik Mühendisliği
- Rossmann veri setini temizledim; eksik değerleri ve aykırı değerleri işledim
- Mevsimsel ve promosyon etkilerini yakalamak için zaman temelli özellikler (gün, ay, promosyonlar, tatiller, rakip mesafesi) oluşturdum

### 🤖 Modelleme
- Mağaza bazında günlük satışı tahmin etmek için XGBoost regresyon modeli eğittim
- Grid search ile hiperparametreleri ayarladım, çapraz doğrulama ile değerlendirdim
- Satışı etkileyen ana faktörleri anlamak için özellik önem analizi yaptım

### 📈 Değerlendirme
- Modeli, tahmin problemlerine uygun regresyon metrikleriyle değerlendirdim: RMSE, MAE ve R²

### İnteraktif Pano
- Herkesin tahminleri interaktif olarak inceleyebilmesi için bir Streamlit uygulaması yayına aldım

### Teknik Araçlar
- **Dil**: Python
- **ML ve veri**: XGBoost, Pandas, NumPy, Scikit-learn
- **Görselleştirme**: Matplotlib, Seaborn
- **Uygulama**: Streamlit
