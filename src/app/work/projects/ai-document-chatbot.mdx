## Project Overview

The Akdeniz Smart Assistant is an AI-powered platform that lets users chat with an organization's documents in natural language. Instead of searching through files manually, a user asks a question and gets an answer grounded in the institution's own documents. It was built by a team of three developers using a modular monolith architecture.

## What the Platform Does

### 🤖 Document understanding & semantic search
- Processes uploaded documents, splits them into segments and turns them into vector embeddings
- Uses semantic (meaning-based) search to find the passages most relevant to a question
- Generates natural-language answers grounded in those passages

### 🔐 Security & access control
- Multi-provider authentication
- Role-based access control (RBAC) so users only see what they are allowed to
- Secured API layer

### 🌐 Multi-language & administration
- Multi-language support
- Admin panel for managing users and content

## Architecture

- **Backend**: Web API built with a modular monolith architecture
- **Search**: Vector-based semantic search over document embeddings
- **Frontend**: TypeScript-based web interface

## Challenges Overcome

- Making semantic document search return relevant, context-aware results
- Designing a real-time chat interface over the retrieval pipeline
- Managing secure, role-based access to sensitive institutional documents

## 🇹🇷 Proje Detayları

Akdeniz Akıllı Asistan, kullanıcıların bir kurumun belgeleriyle doğal dilde sohbet etmesini sağlayan yapay zeka destekli bir platformdur. Kullanıcı, dosyaları tek tek aramak yerine bir soru sorar ve kurumun kendi belgelerine dayanan bir yanıt alır. Üç kişilik bir geliştirici ekibi tarafından modüler monolit mimariyle geliştirildi.

### 🤖 Belge Anlama ve Semantik Arama
- Yüklenen belgeleri işleyip parçalara böldüm ve vektör embedding'lerine dönüştürdüm
- Bir soruyla en alakalı bölümleri bulmak için semantik (anlam temelli) arama kullandım
- Yanıtları bu bölümlere dayandırarak doğal dilde ürettim

### 🔐 Güvenlik ve Erişim Kontrolü
- Çoklu sağlayıcı kimlik doğrulama
- Kullanıcıların yalnızca yetkili oldukları içeriği görmesi için rol tabanlı erişim kontrolü (RBAC)
- Güvenli API katmanı

### 🌐 Çoklu Dil ve Yönetim
- Çoklu dil desteği
- Kullanıcı ve içerik yönetimi için yönetim paneli

### Teknik Mimari
- **Backend**: Modüler monolit mimariyle geliştirilen Web API
- **Arama**: Belge embedding'leri üzerinde vektör tabanlı semantik arama
- **Frontend**: TypeScript tabanlı web arayüzü

### Üstesinden Gelinen Zorluklar
- Semantik belge aramasının alakalı ve bağlam duyarlı sonuçlar döndürmesini sağlamak
- Getirme (retrieval) hattı üzerinde gerçek zamanlı bir sohbet arayüzü tasarlamak
- Hassas kurumsal belgelere güvenli, rol tabanlı erişimi yönetmek
