## Project Overview

An LLM-based Retrieval-Augmented Generation (RAG) chatbot that answers questions about TÜBİTAK support programs. Instead of reading through long program guidelines, users ask a question in natural language and get an answer grounded in the source documents.

## How It Works

### 🔍 Document retrieval
- Splits program documents into chunks and converts them into vector embeddings
- Uses semantic (meaning-based) search to retrieve the passages most relevant to a user's question

### 🤖 Answer generation
- Passes the retrieved passages to a large language model as context
- Generates answers grounded in the source documents, which keeps responses accurate and reduces hallucination

### 🧱 Backend
- Built as a REST API with FastAPI

## 🇹🇷 Proje Detayları

TÜBİTAK destek programları hakkındaki soruları yanıtlayan, LLM tabanlı bir Retrieval-Augmented Generation (RAG) chatbotu. Kullanıcılar uzun program kılavuzlarını okumak yerine doğal dilde soru sorar ve kaynak belgelere dayanan bir yanıt alır.

### 🔍 Belge Getirme
- Program belgelerini parçalara (chunk) bölüp vektör embedding'lerine dönüştürdüm
- Kullanıcının sorusuyla en alakalı bölümleri bulmak için semantik (anlam temelli) arama kullandım

### 🤖 Yanıt Üretimi
- Getirilen bölümleri bağlam olarak büyük dil modeline verdim
- Yanıtları kaynak belgelere dayandırarak doğruluğu artırdım ve halüsinasyonu azalttım

### 🧱 Backend
- FastAPI ile REST API olarak geliştirdim
